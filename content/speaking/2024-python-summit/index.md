## Presentation

- **Topic**: Building Python CLIs in 2024 Does Not Have to Be Painful (feat. Typer and uv)
- **Language**: English
- **Date**: December 2024
- **Format**: Online VoD
- **Co-presenter**: [Mateusz Zaremba](https://www.linkedin.com/in/mateusz-zaremba/), Senior Architect at Ørsted

In a joint talk with [Mateusz Zaremba](https://www.linkedin.com/in/mateusz-zaremba/), we made the case that building command-line tools in Python has gotten dramatically better - and that most developers are still doing it the hard way out of habit.

The talk walked through two tools that change the equation: [Typer](https://typer.tiangolo.com/), a modern CLI framework built on type hints that eliminates most of the boilerplate that made `argparse` painful, and [uv](https://docs.astral.sh/uv/), a blazing-fast Python package and project manager from Astral that removes the friction from dependency management and environment setup.

The presentation covered:

- Why CLI development in Python got a bad reputation - and why that reputation is now outdated
- Typer's type-hint-driven approach to argument parsing, validation, and help generation
- Building a real CLI from scratch: commands, subcommands, options, and autocomplete
- `uv` as a drop-in replacement for the `pip` + `venv` + `pip-tools` workflow
- Combining both tools into a smooth, modern development workflow for distributable CLI tools
- Packaging and publishing a CLI with `uv` so others can install it with a single command

## More

Python Summit is one of the few conferences in Poland fully dedicated to the Python language, organized by the PyData Warsaw and PyWaw communities. The 2024 edition followed the hybrid format: a live online day and an in-person day at the Copernicus Science Centre in Warsaw. Our talk was delivered as a VoD session as part of the online track, reaching around 600 participants.

The summit covers a broad range of Python application areas - from data science and AI to DevOps, web, embedded, and tooling - which made the "tools and best practices" angle a natural fit for the audience.
